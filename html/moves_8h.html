choosing the legal movement.

Parameters
from_plate- the source plate
to_plate- the destination plate

This function pops the last element from both plates. If one element is equal with 0 (the plate is empty), I take the other remaining story and put on the empty plate. Otherwise, choose which element are greater, then push both elements on the other plate, in decreasing order.

implements the movement.

Parameters
from_plate- the source plate
aux_plate- the auxiliary plate
to_plate- the destination plate

The total number of stories is equal with 2 to power the number of stories - 1. Iterating from 1 to total no of moves, this function chooses the case. If number_of_story is even, the interchange to_plate with aux_plate. Since there are 3 circular cases, I use modulo 3.
